# Network-Intrusion-Detection-system-using-XAI
This is a network IDS using XAI (tested and evaluated on CIC IDS Dataset). Firstly the dataset is preprocessed, then the preprocessed dataset is used for training ML models and CNN and LSTM models for comparison. Ensemble model is also created, SHAP and LIME are integrated for explainable AI and a dashboard is created to display the results.
All files are uploaded in txt extension, but are actually saved in .py (python) extension in actual project.
